Εφαρμοφή "MyGreekWay" by BAZINGA
------
Η εφαρμογή έχει ανέβει στο link: https://ellaksrv.datacenter.uoc.gr/~user816/BAZINGA/MyGreekWay.html
Στο φάκελο MyGreekWay περιλαμβάνονται τα αρχεία για την web έκδοση
------

Το πλήρες project του unity μαζί με όλο τον κώδικα βρίσκεται στο φάκελο ProjectX

------
Το αρχείο database.sql περιλαμβάνει την δημιουργία της database για το profile των χρηστών και χαρακτηριστικά των βαλιτσών και αντικειμένων
------

Τα αρχεία στο φάκελο database-read-write περιλαμβάνουν κώδικα για να γίνεται η εγγραφή σε mySQL βάση δεδομένων με html εντολές GET-POST αλλά και η ανάγνωση αυτών σε json format ( ανάκτηση από την sql και μετατροπή σε json ).
